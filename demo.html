Key Features

Block-based Editing

Each piece of content is a separate block, making it easy to organize and structure your documents.

Keyboard Shortcuts

Powerful keyboard shortcuts for efficient editing. Press `/` to access block commands.

Auto-save

Your content is automatically saved to local storage, so you never lose your work.

Responsive Design

Works perfectly on desktop, tablet, and mobile devices with a responsive interface.

Rich Formatting

Support for bold, italic, underline, headings, lists, quotes, and code blocks.

Multiple Pages

Create and organize multiple pages within your workspace for better content management.

Getting Started

Getting started with the editor is simple:

  1. Open the editor - Click the "Try the Editor" button above
  2. Start typing - Click anywhere in the editor and begin typing
  3. Create blocks - Press Enter to create new blocks
  4. Format content - Use the toolbar or keyboard shortcuts
  5. Organize pages - Create new pages in the sidebar

Keyboard Shortcuts

Navigation

  • Enter - New block
  • Backspace - Delete empty block
  • Tab - Indent block
  • Shift + Tab - Outdent block

Formatting

  • Ctrl/Cmd + B - Bold
  • Ctrl/Cmd + I - Italic
  • Ctrl/Cmd + U - Underline
  • Ctrl/Cmd + S - Save

Block Types

The editor supports various block types to help you create rich, structured content:

  • Headings - H1, H2, H3 for document structure
  • Paragraphs - Regular text blocks
  • Lists - Bullet and numbered lists
  • Quotes - Styled quote blocks
  • Code - Monospace code blocks
